## Why

SEO 是个人品牌站的基础优化，确保搜索引擎能正确索引和展示网站内容。通过设置 HTML meta tags、语义化 HTML 和 robots.txt，提升网站的可发现性和专业形象。

## What Changes

- 更新 `index.html` 中的 HTML meta tags（title、description）
- 添加语义化 HTML 审查，确保页面结构符合 SEO 最佳实践
- 添加 `robots.txt` 允许 Google 爬虫索引
- 可选：添加 Open Graph 和 Twitter Card meta tags

## Capabilities

### New Capabilities

- `seo-optimization`: SEO 基础优化，包含 meta tags、语义化 HTML 和 robots.txt

### Modified Capabilities

- 无

## Impact

- 修改 `index.html` 添加 meta tags
- 新增 `public/robots.txt`
- 可能需要审查现有组件的语义化 HTML 结构
